# 尽职调查报告写作规范

> 本文件在 report 模式下加载，指导从底稿到报告的转化。

---

## 一、报告总体结构

```
法律尽职调查报告

一、声明与限定条件
二、目标公司基本信息与主体资格
三、股权结构与股东信息
四、公司治理与组织结构
五、核心资产
六、业务经营与合同管理
七、财务与税务
八、劳动人事管理
九、重大债权债务与担保
十、诉讼、仲裁与行政处罚
十一、其他重要事项
十二、结论与建议
附件：文件清单
```

## 三、声明与限定条件（标准模板）

以下为标准声明模板，根据项目实际情况调整：

### （一）报告出具主体

本报告由[律师事务所全称]（以下简称"本所"）出具。本所接受[委托人全称]（以下简称"委托人"）的委托，就委托人拟[交易目的，如"收购[目标公司全称]（'目标公司'）[X]%股权"]事宜，对目标公司进行法律尽职调查，并出具本法律尽职调查报告（以下简称"本报告"）。

### （二）调查范围

本次法律尽职调查的范围包括但不限于：目标公司的主体资格、股权结构、公司治理、核心资产、业务经营、财务税务、劳动人事、债权债务与担保、诉讼仲裁与行政处罚等方面的法律事项。

本次法律尽职调查的基准日为[日期]（以下简称"基准日"）。除特别注明外，本报告所述的事实和发表的意见均以基准日的状态为准。

### （三）调查方法

本所律师在本次尽职调查过程中采取了以下调查方法：

1. 审阅目标公司及其股东提供的文件资料；
2. 查询国家企业信用信息公示系统、中国裁判文书网、中国执行信息公开网、信用中国等公开信息平台；
3. [如有访谈：对目标公司管理层进行访谈；]
4. 其他本所律师认为必要的调查方法。

### （四）限制条件与免责声明

1. 本报告系根据委托人及目标公司提供的文件资料作出。本所律师已对上述文件资料进行了形式审查，但未对其真实性、准确性和完整性进行独立的实质性验证。如上述文件资料存在虚假、遗漏或误导，可能影响本报告的结论。

2. 本报告仅就法律事项发表意见，不涉及财务、税务、评估等专业事项的判断。涉及财务数据的引用来源于目标公司提供的财务资料，本所律师不对其准确性承担责任。

3. [已说明的重大未获取材料及其影响，例如："由于未能获取[具体文件]，本所无法就[具体事项]作出完整判断。"]

4. 本报告仅供委托人用于[PURPOSE]之用途，未经本所书面同意，不得向第三方披露或用于其他目的。

5. 本报告中的法律意见系基于中国现行有效的法律、法规、规章及规范性文件作出。如前述法律法规发生变化，可能影响本报告的结论。

### （五）经办律师

本次法律尽职调查由本所[律师姓名]律师负责。

## 四、各章节写作规范

### 4.1 章节结构（统一模板）

每章报告内容按以下结构撰写：

```markdown
## [序号]、[章节名称]

### （一）调查范围
[本章调查覆盖的事项，1-2句话概括]

### （二）[主要调查事项名称]
[基本情况描述，客观事实陈述]

### （三）[次要调查事项名称]（如有）
[基本情况描述]

### （四）律师意见
[法律判断，风险提示，建议措施]
```

### 4.2 语言风格要求

**用语原则**：
- 客观、准确、严谨，避免模糊表述
- 使用标准法律术语
- 事实部分：纯描述，不加主观判断
- 意见部分：明确表态，提出具体建议

**确定性表述**（用于明确的事实）：
> 经查，目标公司……；根据目标公司提供的……；经核查……

**限定性表述**（用于信息不完整时）：
> 根据目标公司提供的材料，……；目标公司表示……，本所律师无法独立核实；由于未获取……，本所无法就……作出判断

**风险意见表述**：
> 本所律师注意到……，存在[风险描述]，建议委托人……；
> 如上述[风险事项]实际发生，可能导致……；
> 本所建议在交割前要求目标公司/股东完成……

**应避免的表述**：
- ❌ "我们认为"（改用"本所律师认为"或"本所认为"）
- ❌ "可能有问题"（改为具体描述风险和法律依据）
- ❌ "没有问题"（改为"经查，未发现……"）
- ❌ 口语化表达

## 五、结论与建议章节模板

```markdown
## 十二、结论与建议

### （一）总体结论

经本次法律尽职调查，目标公司[整体合法合规状况的概括评价]。
[对交易整体法律可行性的简要判断]

### （二）重大风险提示

本所律师在本次尽职调查中发现以下重大法律风险事项，提请委托人重点关注：

**1. [重大风险事项1]**
[风险事实]。[法律后果]。建议：[具体建议]。

**2. [重大风险事项2]**
[风险事实]。[法律后果]。建议：[具体建议]。

（按风险等级由高到低列示所有🔴高风险事项）

### （三）交割前建议事项

建议委托人在交割前要求目标公司/转让方解决以下事项：

1. [事项1]
2. [事项2]
3. [事项3]

### （四）交易文件建议

建议在交易文件中纳入以下安排：

1. **陈述与保证**：建议要求转让方就[具体事项]作出陈述与保证；
2. **先决条件**：建议将[具体事项的完成]作为交割先决条件；
3. **特殊赔偿条款**：建议就[具体风险事项]设置专项赔偿安排。
```

## 七、底稿到报告转化要点

| 底稿段落 | 报告对应位置 | 转化说明 |
|----------|-------------|----------|
| §1 调查范围与方法 | 声明（三）调查方法 | 合并各章方法，统一陈述 |
| §2 已获取材料清单 | 附件文件清单 | 汇总为附件，按类别整理 |
| §3 未获取材料 | 声明（四）限定条件 | 重大未获取材料转为免责声明 |
| §4 调查发现 | 各章基本情况 | 客观事实，客户化语言，删去内部工作细节 |
| §5 风险提示 | 各章律师意见 + 结论重大风险 | 🔴高风险进结论章；所有风险项作为律师意见 |
| §6 律师备忘 | **不转入报告** | 完全删除，仅供内部参考 |
