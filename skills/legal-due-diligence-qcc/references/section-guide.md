# 尽调各章节调查指南

## 第1章 公司基本信息与主体资格

### 1.1 调查目的

确认目标公司依法设立并有效存续，具备从事其业务活动的合法主体资格。

### 1.2 调查要点清单

**工商登记信息**：
- [ ] 营业执照（正、副本）：核对公司名称、统一社会信用代码、类型、住所、法定代表人、注册资本、成立日期、营业期限、经营范围
- [ ] 营业执照记载事项与实际经营情况是否一致
- [ ] 营业执照是否在有效期内，是否通过最近一次年检/年报公示

**设立与章程**：
- [ ] 设立批准文件（外资企业、特殊行业等需审批设立的）
- [ ] 公司章程（现行有效版本）
- [ ] 历次章程修正案
- [ ] 章程内容是否符合《公司法》强制性规定
- [ ] 章程中是否存在特殊条款（如限制股权转让的特别约定、一票否决权、优先购买权等）

**变更登记**：
- [ ] 历次变更登记材料（名称、住所、法定代表人、注册资本、经营范围、股东等变更）
- [ ] 变更登记程序是否合法合规（是否经过必要的内部决策程序）
- [ ] 工商登记信息与公司内部文件（章程、决议等）是否一致

**公司治理基础文件**：
- [ ] 股东会、董事会、监事会（监事）议事规则
- [ ] 议事规则是否符合《公司法》和章程规定

**印章与信用**：
- [ ] 公司印章备案情况（公章、合同章、财务章、法人章等）
- [ ] 印章管理制度
- [ ] 企业信用信息公示报告（近一个月内）
- [ ] 是否存在经营异常名录记录
- [ ] 是否存在严重违法失信记录

### 1.3 常见风险点（红旗标志）

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 营业执照过期未续 | 🔴 高 | 主体资格存疑 |
| 经营范围不含实际主营业务 | 🟡 中 | 超范围经营风险 |
| 章程限制股权转让且未获豁免 | 🔴 高 | 直接影响交易可行性 |
| 存在经营异常名录记录 | 🟡 中 | 信用瑕疵，可能影响业务 |
| 工商登记与实际不符 | 🟡 中 | 需核实并整改 |
| 设立程序有瑕疵 | 🔴 高 | 可能影响公司有效存续 |
| 章程中存在不利特殊条款 | 🟡 中 | 需评估对交易的影响 |

## 第2章 股权结构与股东信息

### 2.1 调查目的

核实目标公司的股权结构、各股东出资情况、历次股权变动的合法性，以及股权上的权利限制。

### 2.2 调查要点清单

**股权结构**：
- [ ] 股东名册（现行）
- [ ] 出资证明书
- [ ] 验资报告（各轮次）
- [ ] 历次增资文件（股东会决议、增资协议等）
- [ ] 股权结构图（含穿透至实际控制人）
- [ ] 是否存在代持安排
- [ ] 是否存在员工持股平台

**股权变动**：
- [ ] 历次股权转让协议
- [ ] 股权转让相关股东会决议（其他股东放弃优先购买权的证明）
- [ ] 工商变更登记材料与股权转让协议是否对应
- [ ] 历次股权转让的对价支付凭证

**出资情况**：
- [ ] 各股东实缴出资情况（货币出资银行凭证、非货币出资评估报告）
- [ ] 认缴出资额与实缴出资额
- [ ] 认缴出资期限
- [ ] 是否存在抽逃出资

**股权限制**：
- [ ] 股权质押情况（查询动产融资统一登记公示系统）
- [ ] 股权冻结情况（查询中国裁判文书网、法院执行系统）
- [ ] 对外投资情况
- [ ] 一致行动协议
- [ ] 股东协议中的特殊权利（优先权、反稀释条款等）

### 2.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 存在股权代持 | 🔴 高 | 实际权属不清，存在争议风险 |
| 股权被质押/冻结 | 🔴 高 | 直接影响股权转让的可行性 |
| 出资不实/抽逃出资 | 🔴 高 | 法律责任风险 |
| 历次股权转让程序瑕疵 | 🟡 中 | 可能被原股东主张权利 |
| 认缴出资期限未届满 | 🟡 中 | 需了解加速到期条款 |
| 存在员工持股平台未披露 | 🟡 中 | 影响实际股权比例计算 |

## 第3章 公司治理与组织结构

### 3.1 调查目的

核查目标公司内部治理结构的合法性、有效性，以及重大决策的程序合规性。

### 3.2 调查要点清单

**公司机构**：
- [ ] 现任董事、监事、高级管理人员名单及任期
- [ ] 相关委任文件、股东会/董事会决议
- [ ] 是否存在关联关系（董事与股东、管理层与股东之间）
- [ ] 历次董监高变更记录

**决策文件**：
- [ ] 近三年股东会决议（完整记录）
- [ ] 近三年董事会决议（完整记录）
- [ ] 是否存在应经审批而未经审批的重大事项
- [ ] 重大合同、对外投资、担保等是否经过适当的内部审批程序

**组织架构**：
- [ ] 公司组织架构图
- [ ] 各部门职能说明
- [ ] 子公司/分公司/办事处清单
- [ ] 子公司/分公司的营业执照及基本情况

**内部控制**：
- [ ] 主要内部管理制度（财务、人事、采购、合同等）
- [ ] 是否存在关联交易，及关联交易审批程序

### 3.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 重大决策程序不合规 | 🔴 高 | 相关决议/合同可能被撤销 |
| 实际控制人未在工商登记体现 | 🟡 中 | 影响尽调完整性 |
| 关联交易未经适当审批 | 🟡 中 | 存在被挑战风险 |
| 子公司存在未披露债务/纠纷 | 🔴 高 | 影响整体价值判断 |
| 董监高任职资格存在瑕疵 | 🟢 低 | 一般不影响公司行为效力 |

## 第4章 核心资产

### 4.1 调查目的

核查目标公司核心资产（房产、土地、设备、知识产权等）的权属清晰、无重大瑕疵。

### 4.2 调查要点清单

**不动产**：
- [ ] 自有房产不动产权证书
- [ ] 土地使用权证（出让/划拨性质）
- [ ] 土地/房产是否存在抵押、查封、限制转让
- [ ] 租赁房产的租赁合同（期限、续租权、优先购买权）
- [ ] 租赁房产出租方的权属证明

**动产与设备**：
- [ ] 主要设备清单及购置凭证
- [ ] 融资租赁设备清单
- [ ] 设备是否存在质押

**知识产权**：
- [ ] 商标注册证（覆盖主营业务类别）
- [ ] 专利证书（发明、实用新型、外观设计）
- [ ] 软件著作权登记证书
- [ ] 域名注册信息
- [ ] 商标是否处于有效期内，是否存在争议/无效宣告程序
- [ ] 关键知识产权是否在目标公司名下（而非实际控制人个人名下）

**特许经营与资质**：
- [ ] 行业许可证（食品经营许可、医疗器械经营许可、金融许可等特殊行业）
- [ ] 资质证书（建筑资质、安全生产许可等）
- [ ] 许可证/资质是否在有效期内

### 4.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 核心知识产权在个人名下 | 🔴 高 | 需在交割前完成转让 |
| 土地使用权为划拨性质 | 🟡 中 | 转让需补缴土地出让金 |
| 房产/土地存在抵押或查封 | 🔴 高 | 影响资产处置 |
| 关键许可证即将到期 | 🟡 中 | 需确认续期无障碍 |
| 核心商标未注册或存在争议 | 🔴 高 | 存在品牌权属风险 |
| 软件系统依赖第三方授权 | 🟡 中 | 需核查授权可转让性 |

## 第5章 业务经营与合同管理

### 5.1 调查目的

了解目标公司主要业务模式、核心客户/供应商关系，评估重大合同的合法性和履行风险。

### 5.2 调查要点清单

**业务概况**：
- [ ] 主营业务说明
- [ ] 主要产品/服务介绍
- [ ] 业务资质、行业准入情况

**重大合同**：
- [ ] 合同台账（近三年）
- [ ] 单笔金额超过[阈值]的合同原件
- [ ] 核心客户合同（前5大客户）
- [ ] 核心供应商合同（前5大供应商）
- [ ] 长期合同、独家合同、竞业禁止条款
- [ ] 合同中的变更控制条款（Change of Control Clause）

**特殊业务安排**：
- [ ] 经销商/代理商协议
- [ ] 特许经营协议
- [ ] 战略合作协议
- [ ] 联营/合作协议

**合同履行**：
- [ ] 是否存在重大违约情形
- [ ] 是否存在已过期但仍在履行的合同
- [ ] 是否存在关联方交易合同

### 5.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 重大合同含变更控制条款 | 🔴 高 | 交易触发后可能导致合同终止 |
| 核心客户高度集中 | 🟡 中 | 客户流失风险影响估值 |
| 存在应转让而未转让的合同权利 | 🟡 中 | 需取得相对方同意 |
| 合同约定管辖与实际不符 | 🟢 低 | 可能增加维权成本 |
| 重大合同存在违约风险 | 🔴 高 | 需评估赔偿敞口 |

---

## 第6章 财务与税务

### 6.1 调查目的

核查目标公司财务状况的真实性，识别重大财务风险和税务风险。

### 6.2 调查要点清单

**财务报告**：
- [ ] 近三年经审计财务报告（含审计意见）
- [ ] 最近一期管理账
- [ ] 审计机构资质及独立性

**财务分析**：
- [ ] 主要财务指标（营收、净利润、资产负债率等）
- [ ] 应收账款账龄分析
- [ ] 存货跌价准备
- [ ] 关联方往来款
- [ ] 或有负债（担保、诉讼、合同承诺）

**税务合规**：
- [ ] 税务登记证（现已整合至营业执照）
- [ ] 近三年完税证明/纳税记录
- [ ] 是否存在欠税记录
- [ ] 增值税、企业所得税等主要税种申报情况
- [ ] 是否享受税收优惠（高新技术企业、软件企业等），及优惠是否合规
- [ ] 是否存在税务稽查风险

**特殊财务事项**：
- [ ] 大额资金往来（异常收入、支出）
- [ ] 期末突击入账情况
- [ ] 实际控制人借款情况

### 6.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 审计意见非标准无保留意见 | 🔴 高 | 财务数据可信度存疑 |
| 存在重大欠税记录 | 🔴 高 | 需补缴及滞纳金 |
| 税收优惠不符合条件 | 🔴 高 | 面临补税及处罚风险 |
| 大额关联方往来款未披露 | 🟡 中 | 可能影响财务数据真实性 |
| 应收账款坏账准备不足 | 🟡 中 | 影响实际净资产 |
| 历史偷逃税行为 | 🔴 高 | 追溯补缴风险 |

## 第7章 劳动人事管理

### 7.1 调查目的

核查目标公司劳动用工的合规性，识别劳动争议风险和劳动成本债务。

### 7.2 调查要点清单

**员工情况**：
- [ ] 员工花名册（含岗位、薪酬、入职日期）
- [ ] 劳动合同签订情况（签订率、是否存在未签劳动合同）
- [ ] 固定期限/无固定期限劳动合同比例
- [ ] 关键人员（管理层、技术核心）的合同期限及竞业限制

**社会保险与公积金**：
- [ ] 社保缴纳基数是否合规（是否按实际工资缴纳）
- [ ] 社保参保人数与员工人数是否一致
- [ ] 公积金缴纳情况
- [ ] 历史欠缴社保记录

**薪酬与福利**：
- [ ] 薪酬体系及激励方案
- [ ] 股权激励计划（期权/限制性股票）
- [ ] 年终奖、奖金安排

**用工形式**：
- [ ] 劳务派遣比例（不超过10%）
- [ ] 外包、兼职人员情况
- [ ] 退休返聘人员

**劳动纠纷**：
- [ ] 历史及现存劳动仲裁/诉讼案件
- [ ] 已离职关键人员情况（特别是技术人员）

### 7.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 大量未签劳动合同 | 🔴 高 | 需支付双倍工资赔偿 |
| 历史欠缴社保/公积金 | 🔴 高 | 需补缴，可能有罚款 |
| 劳务派遣比例超标 | 🟡 中 | 整改风险 |
| 关键人员合同即将到期 | 🟡 中 | 留任风险影响估值 |
| 存在集体劳动争议 | 🔴 高 | 需评估赔偿敞口 |
| 股权激励计划未披露 | 🟡 中 | 影响股权结构完整性 |

---

## 第8章 重大债权债务与担保

### 8.1 调查目的

全面梳理目标公司的债权债务状况，识别表外负债和担保风险。

### 8.2 调查要点清单

**债务**：
- [ ] 银行贷款台账（贷款合同、金额、期限、利率、抵押/质押担保情况）
- [ ] 非银金融机构借款（融资租赁、信托等）
- [ ] 股东借款（合同、金额、是否计息）
- [ ] 应付账款账龄分析
- [ ] 其他负债（预收款、税负等）

**对外担保**：
- [ ] 对外提供担保的合同清单
- [ ] 担保金额、期限、被担保方
- [ ] 担保审批程序是否合规

**对外债权**：
- [ ] 应收账款清单
- [ ] 对外借款（含股东借款、员工借款）
- [ ] 预付款回收情况

**或有负债**：
- [ ] 未决诉讼/仲裁的潜在赔偿金额
- [ ] 合同违约金/赔偿金的估算
- [ ] 政府补贴的返还风险

### 8.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 对外担保未披露 | 🔴 高 | 表外负债风险 |
| 贷款存在交叉违约条款 | 🔴 高 | 可能触发全部债务加速到期 |
| 股东借款无还款计划 | 🟡 中 | 影响现金流预测 |
| 大额应付账款逾期 | 🟡 中 | 供应商关系及诉讼风险 |
| 对外担保审批程序不合规 | 🟡 中 | 担保合同可能被撤销 |

---

## 第9章 诉讼、仲裁与行政处罚

### 9.1 调查目的

全面了解目标公司涉及的法律纠纷和行政处罚情况，评估潜在法律风险敞口。

### 9.2 调查要点清单

**诉讼与仲裁**：
- [ ] 查询中国裁判文书网（以公司名称为当事人）
- [ ] 查询中国执行信息公开网（被执行人信息）
- [ ] 查询各地法院公告（开庭公告）
- [ ] 公司提供的诉讼清单（含已结案和在案）
- [ ] 重大诉讼/仲裁的案情、标的金额、当前状态、律师意见

**行政处罚**：
- [ ] 市场监管部门处罚记录（国家企业信用信息公示系统）
- [ ] 行业主管部门处罚记录（如环保、税务、劳动、食药监等）
- [ ] 处罚原因、金额、是否已整改

**合规性**：
- [ ] 是否持有必要的行政许可
- [ ] 是否存在行政处罚记录影响许可续期
- [ ] 是否存在刑事风险（如经营者涉及刑事案件）

### 9.3 常见风险点

| 风险事项 | 风险等级 | 说明 |
|----------|----------|------|
| 存在未披露的重大诉讼 | 🔴 高 | 影响估值和交割条件 |
| 被列为被执行人 | 🔴 高 | 信用严重受损，资产可能被查封 |
| 重大行政处罚影响许可续期 | 🔴 高 | 可能影响经营持续性 |
| 存在同类诉讼多起 | 🟡 中 | 系统性风险信号 |
| 创始人/实控人涉及刑事调查 | 🔴 高 | 需谨慎评估交易风险 |

## 第10章 其他重要事项

### 10.1 调查目的

对上述九章未覆盖但对交易有重要影响的事项进行补充调查。

### 10.2 调查要点清单

**数据合规（适用于数据密集型企业）**：
- [ ] 个人信息保护合规情况（隐私政策、用户协议）
- [ ] 数据安全保护措施
