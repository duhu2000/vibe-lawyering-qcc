## 🚀 快速开始

### 1. 克隆本仓库

```bash
git clone https://github.com/your-org/vibe-lawyering-qcc.git
cd vibe-lawyering-qcc
```

### 2. 配置企查查MCP

```bash
# 设置环境变量
export QCC_MCP_API_KEY="your_api_key_here"

# 验证配置
source ./scripts/setup-qcc-env.sh
```

从 [企查查智能体数据平台](https://agent.qcc.com) 申请API Key。

### 3. 启动Claude Code

```bash
claude --plugin-dir .
```

### 4. 体验增强SKILL

| 场景 | Command | 企查查增强 |
|------|---------|-----------|
| 法律尽职调查 | `/legal-dd-qcc 企查查科技` | ✅ 自动获取工商、股权、风险、知产数据 |
| 合同审查 | `/contract-review-qcc 合同.docx --party XX公司` | ✅ 交易对手风险核查 |
| 法律研究 | `/legal-research-qcc` | ✅ 关联企业案例检索 |

### 场景2：合同审查 + 交易对手六维风险核查

**使用Command**: `/contract-review-qcc 采购合同.docx --party 北京XX科技有限公司`

```
📝 用户输入：
"请审查这份采购合同，对方是北京XX科技有限公司"

⚡ 自动执行：
├── SKILL：contract-review-qcc 启动合同审核
├── 企查查MCP六维风险核查：
│   ├── qcc-company: 主体真实性 + 股权稳定性
│   │   ├── 企业状态：存续 ✅
│   │   ├── 注册资本：1000万（实缴200万）⚠️
│   │   ├── 成立时间：2021年（成立不满3年）⚠️
│   │   ├── 股权结构：股东A 60%、股东B 40%
│   │   ├── 股权质押：无 ✅
│   │   └── 股权冻结：无 ✅
│   ├── qcc-risk: 信用风险 + 诉讼风险 + 经营风险
│   │   ├── 失信记录：0条 ✅
│   │   ├── 被执行：2条（金额50万）🔴
│   │   ├── 限高记录：1条 🔴
│   │   ├── 司法诉讼：5件（作为被告4件）⚠️
│   │   └── 经营异常：1条（地址变更未备案）🟡
│   └── qcc-operation: 资质风险
│       ├── 行业资质：XXX资质证书 ✅
│       ├── 资质有效期：202X年到期 ⚠️
│       └── 近一年招投标：3条 ✅
├── 合同条款审查
└── 生成修订批注版docx + 交易对手六维风险报告

📄 输出结果：
    【交易对手六维风险报告 - 北京XX科技有限公司】

    【维度1/6 - 主体真实性】qcc-company
    ✅ 企业状态：存续（在营）
    ⚠️  注册资本：1000万（实缴仅200万，实缴率20%）
    ⚠️  成立时间：2021年（成立不满3年，履约能力待观察）

    【维度2/6 - 股权稳定性】qcc-company
    ✅ 股权结构：清晰（股东A 60%、股东B 40%）
    ✅ 股权质押：无
    ✅ 股权冻结：无
    ✅ 近一年股权变更：0次

    【维度3/6 - 信用风险】qcc-risk
    🔴 被执行记录：2条（累计金额50万）
    🔴 限高记录：1条（法定代表人被限高）
    ✅ 无失信记录

    【维度4/6 - 诉讼风险】qcc-risk
    ⚠️  司法诉讼：5件
        - 作为被告：4件（占比80%）⚠️
        - 作为原告：1件
    ⚠️  最近诉讼：202X年X月（距今X个月）

    【维度5/6 - 经营风险】qcc-risk + qcc-operation
    🟡 经营异常：1条（地址变更未及时备案，已整改）
    ✅ 无行政处罚记录
    ✅ 近一年经营动态：正常

    【维度6/6 - 资质风险】qcc-operation
    ✅ 特殊资质：持有XXX资质证书
    ⚠️  资质有效期：至202X年X月（距今不足1年）

    【综合评估】
    🔴 整体风险等级：高风险
    ⚠️  主要风险：被执行记录、限高记录、作为被告比例高

    【合同审查 - 交易对手风险批注】
    问题：交易对手存在多项风险信号
    风险：
    - 被执行记录可能影响履约能力
    - 法定代表人被限高可能影响商务活动
    - 作为被告案件占比高，应诉压力大
    修改建议：
    1. 增加履约保证金条款（建议合同金额20%）
    2. 增加分期付款安排（3:4:3比例）
    3. 要求提供第三方担保或母公司连带保证
    4. 增加违约责任条款（逾期按日千分之五计算）
```
