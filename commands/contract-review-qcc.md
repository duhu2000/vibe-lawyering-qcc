# /contract-review-qcc 命令

## 功能

审查中文合同条款，企查查MCP增强版自动核查交易对手企业信用风险、司法诉讼、股权稳定性，输出逐条批注修订版docx + 交易对手风险报告。

## 使用方法

```
/contract-review-qcc 采购合同.docx --party 北京XX科技有限公司 --stand 甲方
```

## 参数说明

| 参数 | 必需 | 说明 |
|------|------|------|
| `合同文件` | ✅ | 合同文件路径（docx/md/txt） |
| `--party` | 可选 | 交易对手企业全称（用于企查查核查） |
| `--stand` | 可选 | 审查立场：甲方/乙方/中性（默认中性） |
| `--template` | 可选 | 参考模板文件路径 |

## 自动执行流程

1. **合同读取**：解析合同文本、识别条款
2. **交易对手识别**：提取合同中的企业名称
3. **企查查风险核查**（如提供企业名称）：
   - `qcc-company`: 工商信息、股权结构
   - `qcc-risk`: 失信、被执行、司法诉讼
   - `qcc-operation`: 经营异常、行政处罚
4. **合同审查**：逐条审核条款
5. **风险批注生成**：合同风险 + 交易对手风险
6. **输出文件**：
   - 合同名称-修订批注版.docx
   - 交易对手风险报告-[企业名称].md

## 输出文件

```
合同名称-Output/
├── 合同名称-修订批注版.docx      # 带修订痕迹和批注
└── 交易对手风险报告-[企业].md    # 企查查风险数据（如有）
```

## 企查查风险核查维度

| 维度 | 说明 |
|------|------|
| 主体真实性 | 企业存续状态、注册资本真实性 |
| 股权稳定性 | 股东结构、股权质押/冻结 |
| 信用风险 | 失信记录、被执行记录 |
| 诉讼风险 | 涉诉案件、作为被告比例 |
| 经营风险 | 经营异常、行政处罚 |

## 示例

```
/contract-review-qcc 服务合同.docx --party 上海XX信息科技有限公司 --stand 甲方

→ 读取合同文件
→ 调用企查查MCP核查上海XX信息科技有限公司
→ 发现：被执行2条、司法诉讼5件
→ 在合同中增加交易对手风险批注
→ 建议增加履约保证金条款
→ 生成修订批注版docx + 风险报告
```
